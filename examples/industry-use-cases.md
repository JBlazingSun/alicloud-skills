# 行业场景 × Skills 组合（可落地需求清单）

本文档汇总基于当前项目 skills 可落地的行业场景需求，并标注对应技能组合。

## 互联网

- 新品发布内容自动生成图/视频/音频并一键上线可访问
  - skills: image + video + audio + fc + dns
- 对外提供产品发布信息 API 给第三方展示
  - skills: fc
- 素材自动打包存 OSS 并给团队下载
  - skills: image + video + audio + oss

## 电商

- 新品上架多素材包（主图/短视频/卖点语音）
  - skills: image + video + audio
- 活动落地页绑定子域名可直接访问
  - skills: fc + dns
- 商品素材统一存 OSS，提供清单
  - skills: oss + image + video + audio

## 新媒体

- 热点事件快速生成全套专题素材并上线
  - skills: image + video + audio + fc + dns
- 生成可投放新闻短视频 + 播客音频
  - skills: video + audio
- 结构化事件 JSON 输出给分发平台
  - skills: fc

## 教育培训

- 课程上线素材：课程预告图、试听视频、讲解音频
  - skills: image + video + audio
- 课程介绍页可被分享访问（自定义域名）
  - skills: fc + dns
- 课程素材统一入库 OSS，便于团队分发
  - skills: oss + image + video + audio

## 金融/保险

- 每周市场解读：摘要图 + 讲解音频 + 简短视频
  - skills: image + audio + video
- 对外提供结构化报告 API（含免责声明字段）
  - skills: fc
- 客户访问页，用自定义域名避免下载
  - skills: fc + dns

## 制造工业

- 产品发布包：性能参数图解 + 工厂短片 + 语音说明
  - skills: image + video + audio
- 合作方可访问的介绍页 + 子域名
  - skills: fc + dns
- 宣传素材统一打包 OSS
  - skills: oss + image + video + audio

## 游戏

- 新游预约活动素材包（KV/预告片/旁白）
  - skills: image + video + audio
- 活动落地页 + 子域名
  - skills: fc + dns
- 输出给渠道的活动 API
  - skills: fc
